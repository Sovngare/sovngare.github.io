Инструкция по установке

  1. Скачать проект по ссылке ниже
  2. Распаковать архив
  3. Открыть папку TrainKids -> TrainKids -> bin -> Debug
  4. Запустите TrainKids.exe
  5. Для работы с приложением требуется пройти процедуру аутентификации или регистрации
    Данные тестового аккаунта:
